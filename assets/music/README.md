# Background music

Place your music files here. The game will play:

- **menu.ogg** / **menu.mp3** — main menu (or **background.*** if missing)
- **battle.ogg** / **battle.mp3** — battle scene (or **background.*** if missing)
- **background.ogg** / **background.mp3** — single track used for both menu and battle if the above are missing

Supported formats: **OGG Vorbis**, **MP3**.

**Example:** put your file here and name it **background.mp3** — it will play in the menu and in battle.
